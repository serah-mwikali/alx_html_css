# SmileSchool - CSS Advanced Project

## Project Overview
This project is a responsive website for SmileSchool, a platform for learning smiling techniques. The project focuses on implementing advanced CSS techniques based on a provided Figma design.

## Learning Objectives
- Implement advanced CSS positioning and layout techniques
- Create responsive designs using media queries
- Use CSS variables for consistent theming
- Apply modern CSS features like Flexbox and Grid
- Follow design specifications from Figma mockups

## Design Resources
- **Figma Design**: [SmileSchool Design](https://www.figma.com/file/your-design-link)
- **Fonts**: Source Sans Pro and Spin-Cycle-OT
- **Images**: Downloaded from project resources

## Setup Instructions
1. Clone the repository
2. Ensure all images are in the images/ folder
3. Open index.html in a web browser
4. The website should render according to the Figma design
